# Portfolio

---

## 🚀 Description

A modern videography portfolio showcasing cinematic storytelling, brand visuals, and creative video production. This platform highlights my work across commercials, events, and digital content, focusing on high-quality visuals, smooth editing, and compelling narratives that help brands and individuals stand out.

---

## ✨ Features

🎥 Video showcase (optimized WebM format)
⚡ Smooth scrolling experience (Lenis integration)
🎨 Modern UI with Tailwind CSS
📱 Fully responsive design
🎯 Clean section-based navigation (About, Portfolio, Contact)
🔗 WhatsApp integration for direct client communication
🎬 Cinematic gradients and visual effects
⬆️ Scroll-to-top interaction

---

## 🛠 Tech Stack

Frontend: React (Vite)
Styling: Tailwind CSS
Animations & Scroll: Lenis
Routing: React Router
Icons: Lucide / SVG
